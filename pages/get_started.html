GET STARTED GUIDE
  1. Create your html in the pages directory
    add your html page name in main.js in the pageNames array
    (the pageLoader will load these pages for your)
  2. Create your js files/class in js directory
    Add the js script at the end of index.html
    There is a section in main.js for global variables
  3. In main.js, add your initialize function including any data loads
    Also add the name of your function to loadData()
  4. The only common pages should be:
    index.html: to add any JS scripts
    main.js: to add any global variables
    main.js: to initialize your vis and load data
  5. Styling
    Try to use bootstrap styles when possible
    We can decide on a common style template for all pages (copy/paste one of the templates I created for now
    Remember to use id="xxx" for anything custom (and add to the style.css)
